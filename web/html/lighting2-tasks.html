Pass the labyrinth

Get to the end of the labyrinth. But this time it looks like all walls are painted in the beautiful color of Hamburg's sky.
So good luck!
Or, well, you could get some depth in here.

  1. You should already have the ambient light implemented.
  2. Open the file
    shaders/tasks/vertex_lighting.vs
    . Notice that all required light source attributes and uniforms are already passed to the shader.
  3. Calculate the vectors V, N, L and R. These are some useful GLSL functions to keep in mind:
    • normalize()
    • reflect()
  4. Calculate the the light attenuation factor fAtt.
    These are some useful GLSL functions to keep in mind:
    • distance()
    • min()
    • pow()
  5. Implement the diffuse light based on the Phong lighting model in the file
    shaders/tasks/vertex_lighting.vs
    .
    These are some useful GLSL functions to keep in mind:
    • max()
    • dot()
  6. Pass the labyrinth.
