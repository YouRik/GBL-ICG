Pass the labyrinth

Get to the end of the labyrinth. This time we've got some shiny material on the walls with only a specular coefficient.

  1. You should already have the ambient and diffuse light implemented in the fragment shader.
  2. Open the file
    shaders/tasks/fragment_lighting.vs
    . Notice that all required light source attributes and uniforms are already passed to the shader.
  3. Implement the specular light based on the Phong lighting model. Make sure to calculate the vectors V, N, L and R correctly.
    These are some useful GLSL functions to keep in mind:
    • normalize()
    • reflect()
  4. Pass the labyrinth.
