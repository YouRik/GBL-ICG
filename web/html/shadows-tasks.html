Great, the orb we need is behind that glass. To open it, bring the left orb to the blue pedestal. But to keep it cool on the way, we will need shadows.

  1. Open the file
    web/common/Game.js
    . Note that the shadow/depth map is already almost fully implemented in lines 304 onwards. The render loop can be looked up in the same file
  2. Finish the initialization as described in the comment marked with TASK4.1. You will need the following function:
    • GL.framebufferTexture2D()
  3. Open the file
    web/shaders/tasks/depth.vs
    and complete the missing transformation to light space.
  4. Then open the file
    web/shaders/tasks/depth.fs
    and set the depth to a value that makes the whole shadow map appear in shadow such that the orbs get cooled down.
  5. Open the file
    web/shaders/tasks/fragmentLighting.vs
    and implement the missing transformation.
  6. Now open the file
    web/shaders/tasks/fragmentLighting.fs
    and follow the remaining tasks marked with TASK4.1
  7. Come back here and bring the first orb to the blue pedestal!

Perfect. Now we need more accurate shadows to keep the second orb in a good temperature and get it to the gate.

  1. Open the file
    web/shaders/tasks/depth.fs
    and set the fragment's depth correctly. You will need this built-in variable:
    • gl_FragCoord
  2. Then open the file
    web/shaders/tasks/fragmentLighting.fs
    and follow the remaining tasks marked with TASK4.2. This function will be of use:
    • texture()
  3. Come back here and carry the orb to the gate using the cloudy path.
