Get to the end of the labyrinth. But as you can see... you can't see...
So good luck!
Or, well, you could just get some light in here.

  1. Open the file
    shaders/tasks/vertex_lighting.vs
    . Notice that all required attributes and uniforms are already defined in the shader.
  2. Before you can use these variables in the shader, the correct values need to be passed. Open the file
    common/GameObject.js
    and get the correct uniform locations in the constructor.
    You will need the following WebGL function:
    GL.getUniformLocation()
  3. Now you need to pass the uniform values to the correct locations in the shader in render method.
    You will need the following WebGL function:
    GL.uniform[1234][fi][v]()
  4. Implement the ambient light in the shader based on the Phong lighting model in the file
    shaders/tasks/vertex_lighting.vs
    .
  5. Pass the labyrinth.
